# krishnacraftcode.com

This repository contains the source code for the Krishna Craft Code website.

- Craft products listing
- User authentication
- Order management
- Responsive UI

## Tech Stack

- MySQL

- JavaScript

## Setup
1. Clone repo:
   git clone https://github.com/username/krishnacraftcode.com.git
2. Install dependencies
3. Configure `.env`
4. Run migrations
5. Start server
